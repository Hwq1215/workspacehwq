  • 和之前那一样是Code, Msg, Data 封装的json数据

登录

  • 接口：
    @GetMapping("/login/in")
  • 传到后端的数据：
    FormData{
        userId: int
        password : string
    }
  • 传回前端的数据：
    code{}
    msg{}
    data{null}
状态 code
用户存在 200
密码错误 210
用户不存在 220


使用说明

  • 接口：
    @GetMapping("/instruction")
  • 传到前端的数据：
    data{
       imgPath: string //是一个图片名字，和之前一样,你需要在前端拼接 http://localhost:8081/
    }


个人信息修改

  1. 获取用户信息的接口：
    @GetMapping("user/get")
    //带上userId
  • 传到前端的数据：
    data{
        userId: int
        password : string
    }
  1. 修改用户信息的接口：
    @PostMapping("user/change")
  • 传到后端的数据：
    FormData{
        userId: int
        password : string
    }

*传回前端的返回值：

状态 code
修改成功 200
用户不存在 210
修改失败 404
